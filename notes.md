# Notes for ChromeGPT Extension

## Installation

1. Clone this repository to your local machine.
2. Navigate to the `ChromeGPT` directory in your terminal or command prompt.
3. Install the necessary dependencies using npm:
   ```bash
   npm install
   ```
4. Create a new service worker by copying the `backgroundServiceWorker.js` file and renaming it to `backgroundServiceWorker.js`.

## Configuration

1. Replace `YOUR_API_KEY` in the `background.js` file with your actual OpenAI API key.
2. Customize the `popup.html` file to fit your needs.

## Usage

1. Install the extension from the Chrome Web Store.
2. Click the extension icon to start using it on any website where you want to fill in forms.
