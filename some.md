## Social Media Extension: Instagram Comment Sentiment Analysis
Description:

With this social media extension, users can read all comments on any Instagram post and analyze their sentiment. The extension will automatically gather the text from all comments and then analyze the sentiment of each comment using natural language processing (NLP).

## Main Features:

Read All Comments: Users can access all comments from a specific Instagram post.
Sentiment Analysis: The extension will analyze the sentiment of each comment using the local Ollama instance.
Gather Links: If available, the extension will also gather all links from each comment and provide additional context.

## Usage:

Install the Extension: Download and install the extension on your browser.
Select a Post: Click on an Instagram post to see all comments.
Read Sentiment Analysis: The extension will display the sentiment analysis of each comment along with the text.
Gather Links: If available, the extension will also show links associated with each comment.
How It Works:

Fetch Comments: When a user selects a post, the extension will fetch all comments from that post using the Instagram API.
Local Ollama Instance: The extension will use a local Ollama instance to analyze the sentiment of each comment. This can be done using libraries like textblob or spaCy.
Display Results: The results will be displayed in a user-friendly interface, including the text, sentiment analysis, and any relevant links.
## Notes:

This extension requires an active internet connection to fetch comments from Instagram.
The sentiment analysis is based on machine learning models and may not always provide accurate results.
The extension does not store any sensitive information, such as personal data or user names, unless explicitly requested by the user.
## Additional Features:

Custom Filters: Users can add custom filters to exclude certain words or phrases from the sentiment analysis.
Search Filter: Users can search for specific keywords or phrases in the comments.
Notification Alerts: The extension can send notifications when new comments are added to a post.
## Conclusion:

This social media extension provides valuable insights into user interactions on Instagram, which can be useful for businesses and marketers looking to understand customer sentiment and behavior.
