# ChromeGPT User Guide

## Introduction

ChromeGPT is a local-first AI form assistance extension for browser-based applications. It allows you to use AI-generated responses and generate custom prompts based on user input.

## Installation

1. **Download the Extension**:
   - Go to [the Chrome Web Store](https://chrome.google.com/webstore/developer/) and search for "ChromeGPT".
   - Click on the "Add to Chrome" button to install the extension.

2. **Enable the Extension**:
   - Once installed, click on the extension icon in the action menu or the toolbar.
   - Enable the extension by clicking on the toggle switch.

## Usage

### Hovering Over Text Boxes

1. **Hover over a text box**: When you hover over a text box in any browser-based application, ChromeGPT will appear as a drop-down list with options to use AI-generated responses and generate custom prompts.

2. **Select an Option**: Choose the option that best suits your needs and click on it to interact with the extension.

### Creating Files

1. **Open the Chat UI**: Hover over a text box and select the "Chat" option to open the chat UI.

2. **Upload a File**: In the chat UI, you can upload files using the file picker.

3. **Write in the Chat Window**: Type your message or prompt in the chat window.

4. **Continue**: Click on the "Continue" button to continue the conversation with AI-generated responses and generate custom prompts.

### Additional Features

- **Custom Prompt Generation**: You can generate custom prompts based on user input by typing a keyword followed by a question mark.
- **File Writing Tools**: ChromeGPT provides file-writing tools that allow you to create files using AI-generated text.
